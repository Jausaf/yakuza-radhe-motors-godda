# ⚡ YAKUZA Website - Quick GitHub Deployment (5 Minutes)

## 🚀 Super Quick Steps

### 1️⃣ Create GitHub Account
- Go to **github.com**
- Click **Sign up**
- Complete registration

### 2️⃣ Create Repository
- Click **+** icon → **New repository**
- Name: `yakuza-electric-scooters`
- Select **Public**
- Click **Create repository**

### 3️⃣ Upload Website
- Click **Add file** → **Upload files**
- Drag your HTML file
- **IMPORTANT:** Rename to `index.html`
- Click **Commit changes**

### 4️⃣ Enable GitHub Pages
- Go to **Settings**
- Find **Pages** (left menu)
- Select **Branch: main**
- Click **Save**

### 5️⃣ Your Website is LIVE! 🎉
```
https://YOUR-USERNAME.github.io/yakuza-electric-scooters
```
